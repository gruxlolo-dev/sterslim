# 🚀 sterslim

Lightweight PHP Starter Kit.

---

## 📦 Install

```bash
composer create-project gruxlolo-dev/sterslim my-app --repository='{"type":"vcs","url":"https://github.com/gruxlolo-dev/sterslim"}' -s dev
```

or manual:

```bash
git clone ...
cd sterslim
bash installer/install.sh
```

---

## ⚙️ Features

- Slim routing
- PSR-4 structure
- dotenv support
- clean architecture

---

## 📜 Description
Sterslim is an open-source PHP starter kit built on Slim framework. It provides a ready-to-use project structure, Docker configuration, and modern development tools to quickly bootstrap scalable PHP applications.
