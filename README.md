# 🚀 sterslim

Lightweight PHP Starter Kit.

---

## 📦 Install

```bash
composer create-project kacper5drub/sterslim my-app
```

or manual:

```bash
git clone ...
cd sterslim
bash installer/install.sh
```

---

## ⚙️ Features

- Slim routing
- PSR-4 structure
- dotenv support
- clean architecture
